Project Setup

1)Install the Angular CLI

npm install -g @angular/cli

2)Create a workspace

ng new my-app

3)Run the application

  i)Navigate to the workspace folder,
   such as my-app.
  ii)Run the following command:
   
    cd my-app
    ng serve --open

  Difference between subject and observable
  
  • subject: you can send to it and receive from it.
    • 
    
  •  Observable: you can receive from it only.
    • 
    
    
      
    
       
    
        Subject
        only upcoming values
       
    
       
    
        BehaviourSubject
        ...one previous value and upcoming values
       
    
       
    
        ReplaySubject
        ...all previous value and upcoming values
